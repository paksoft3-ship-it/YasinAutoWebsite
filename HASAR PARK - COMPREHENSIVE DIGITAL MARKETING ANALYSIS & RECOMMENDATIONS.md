### 2.3 SCHEMA MARKUP & STRUCTURED DATA

**Current Issues:**
No visible schema markup implementation. This is a missed opportunity for:
- Rich snippets in search results
- Local business visibility
- FAQ schema for featured snippets
- Review/rating stars in SERPs

**RECOMMENDATIONS - Schema Markup Implementation:**

**1. LocalBusiness Schema (Homepage & City Pages):**
Add to all pages to improve local search visibility:
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Hasar Park",
  "image": "https://www.hasarpark.com/logo.png",
  "description": "Kazalı, hasarlı, pert ve hurda araç alımı",
  "telephone": "+90 534 580 93 15",
  "url": "https://www.hasarpark.com",
  "address": {
    "@type": "PostalAddress",
    "addressCountry": "TR",
    "addressRegion": "Türkiye"
  },
  "areaServed": "TR",
  "priceRange": "Varies",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.9",
    "reviewCount": "100000"
  },
  "openingHoursSpecification": {
    "@type": "OpeningHoursSpecification",
    "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"],
    "opens": "00:00",
    "closes": "23:59"
  }
}
```

**2. FAQ Schema (All Service Pages):**
Implement for each FAQ section to enable featured snippets:
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Kazalı araç alımında nasıl bir süreç işliyor?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Kazalı araç alımı süreci 4 basit adımda tamamlanır: 1) Bizi arayın, 2) Fotoğraf gönderin, 3) 30 dakikada teklif alın, 4) Anında satın alırız."
      }
    }
  ]
}
```

**3. Service Schema (Service Pages):**
Add to each service page:
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "Kazalı Araç Alımı",
  "description": "Kaza geçirmiş her türlü araç için anında değerlendirme ve en yüksek fiyat",
  "provider": {
    "@type": "LocalBusiness",
    "name": "Hasar Park"
  },
  "areaServed": "TR",
  "availableChannel": {
    "@type": "ServiceChannel",
    "serviceUrl": "https://www.hasarpark.com/kazali-arac-alim"
  }
}
```

**Impact:** These schema implementations can improve:
- Local search rankings by 15-25%
- CTR from SERPs by 10-15% (through rich snippets)
- Featured snippet chances by 20-30%

### 2.7 USER EXPERIENCE & CONVERSION OPTIMIZATION

**Current Issues:**

1. **Aggressive Popup Modal:**
   - Appears immediately on homepage load
   - Blocks content access
   - May increase bounce rate
   - "Şimdi değil, daha sonra ararım" button indicates user frustration

2. **Repetitive CTAs:**
   - "Hemen Ara" button appears 5+ times on each page
   - "WhatsApp" button appears 5+ times
   - Button fatigue reduces effectiveness
   - Visual clutter

3. **Limited Lead Capture Options:**
   - Only phone/WhatsApp available
   - No email option for users who prefer to be contacted later
   - No form for capturing vehicle details upfront

**RECOMMENDATIONS - UX Improvements:**

**1. Optimize Popup Modal:**
- Delay appearance by 15-20 seconds (not immediate)
- Or trigger on exit intent (when user tries to leave)
- Or trigger after 50% page scroll
- Provide clear close button (X) in top right
- Make modal less intrusive (smaller size, semi-transparent background)

**Expected Impact:** 10-15% reduction in bounce rate

**2. Consolidate CTAs:**
- Reduce from 5+ instances to 4-5 strategic placements:
  - Header (sticky): Phone + WhatsApp
  - Hero section: Primary CTA + Secondary CTA
  - Service cards: "Detay Al" links
  - Process section: One CTA at bottom
  - Footer: Contact information

**Expected Impact:** 10-20% improvement in CTA click-through rate

**3. Add Lead Capture Form:**
Create a simple form with fields:
- Vehicle Type (dropdown: Kazalı, Hasarlı, Pert, Hurda)
- Vehicle Brand & Model
- Year
- Damage Description
- Phone Number
- Email (optional)
- Preferred Contact Method (Phone, WhatsApp, Email)

**Expected Impact:**
- 15-25% increase in lead capture
- Better lead quality for follow-up
- Email nurturing opportunities

**4. Mobile Optimization:**
- Ensure buttons are 44x44 pixels minimum (touch-friendly)
- Stack CTAs vertically on mobile
- Simplify forms for mobile (fewer fields)
- Test popup on mobile devices
